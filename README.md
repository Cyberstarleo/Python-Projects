# Python_Customer_Analysis
Customer Data Analysis —
A simple Python-based analysis to understand customer demographics and spending behavior, helping businesses make smarter pricing and marketing decisions.
 
Key Insights
Average customer age: 49 → Primary target group.
Average spend: ~280 → Ideal pricing: 280–300.
Highest spenders: Ages 30–50.
Non-married customers spend more.
Arizona generates the most profit.
Male customers have the highest average spending.
 
Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

Conclusion
The analysis highlights where the business should focus — the right age group, pricing range, and customer segments — helping improve overall sales and profitability.
